# Instagram予約投稿ツール — One‑Pager

## Why (背景・課題)
- インフルエンサーや中小企業は **定期的な投稿** が必須だが、時間・人手が不足し投稿頻度が不安定。
- 既存の無料ツール（Meta Creator Studio 等）は柔軟性不足。
  海外製（Buffer 等）は日本語未対応、国内大型ツール（Social Insight 等）は高価格。
- **手軽かつ安全に“投稿の自動化”を実現するサービス空白** が市場に残っている。

## Who (ターゲット)
- **個人インフルエンサー**：副業クリエイター、フォロワー拡大を狙う個人。
- **中小規模事業者**：美容室・カフェ・ECショップなど、集客のためにInstagram運用が不可欠だが専任担当者がいない。
- 共通ニーズ：低コスト・日本語UI・直感的操作・投稿失敗ゼロの安心感。

## What (解決策・プロダクト概要)
- **Instagram公式API** 連携による安全な予約投稿プラットフォーム。
- 柔軟な投稿オプション
  1. **日時指定予約**
  2. **曜日×時刻の繰り返し投稿**
  3. **同一コンテンツの再投稿**
  4. **登録コンテンツからランダム投稿**
- カレンダーUI＋ライブラリ管理で「作り貯め → 自動投稿」を実現。
- スマホ対応・シンプル操作。

## How (差別化戦略・実現方法)
- **機能差別化**：繰り返し／ランダム投稿は競合未対応。
- **価格戦略**：フリーミアム＋低価格有料プラン（月額数千円）。
- **日本語UX & サポート**：英語ツールの障壁を除去。
- **技術基盤**：Firebase Cloud Functions + Firestore + Stripe。
  - 自動スケールで数千アカウントも対応。
  - Meta公式APIで規約違反リスクゼロ。

## Success (成功指標)
| 期間 | KPI | 目標値 |
|------|-----|--------|
| 6か月 | 月間アクティブユーザー (MAU) | 500 |
|      | 1ユーザー平均予約投稿数 | 15件/月 |
| 12か月 | 有料プラン転換率 | 20 % |
|      | NPS (ユーザー満足度) | +40 |

成功事例を創出：
- 「定期投稿化でフォロワー +150%」
- 「自動投稿で運用時間 70% 削減」
